## 🧠 Contexto

Um e-commerce em crescimento apresenta bons números de faturamento e volume de pedidos, mas esconde desafios importantes:

- Baixa retenção de clientes  
- Forte concentração de receita em poucos estados  
- Alta variabilidade nos valores de pedidos  
- Dependência de categorias específicas  

📌 **Pergunta central:**  
O crescimento é sustentável ou depende exclusivamente da aquisição constante de novos clientes?

---

## 🎯 Objetivo

Este projeto tem como objetivo analisar dados de vendas de um e-commerce para extrair insights relevantes que auxiliem na tomada de decisão, como:

- Performance de faturamento  
- Comportamento de compra  
- Distribuição geográfica  
- Retenção de clientes  
- Análise de produtos

## 📅 Análise Temporal

### 📈 Tendência de faturamento

- Forte crescimento ao longo de 2017  
- Pico entre **fevereiro e agosto de 2018**  
- Queda brusca nos meses finais  

📌 Existe uma **sazonalidade clara**, com concentração de vendas no meio do ano.

## 🔍 Principais Insights

- Faturamento concentrado no Sudeste e Sul
